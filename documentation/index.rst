Documentation
=============

`GlobaLeaks <https://www.globaleaks.org/>`_ is an open-source, free software intended to enable anyone to easily set up and maintain a secure whistleblowing platform.


.. note:: This documentation is thank to the support of our community. Join us and contribute with your additions and suggestion. In any of the page you find a link that enables you to provide suggestions and corrections. We remind you that in case of any software issue or bug you may always report on the `ticketing system <https://github.com/globaleaks/whistleblowing-software/issues>`_.
